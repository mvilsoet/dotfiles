i3wm Keybindings:

# Launch and Exit
Mod+Enter        → Open terminal
Mod+d            → Run dmenu
Mod+space        → Launch rofi (run launcher)
Mod+Shift+space  → Toggle floating mode (popup cheatsheet here!)
Mod+Shift+c      → Reload i3 config
Mod+Shift+r      → Restart i3 inplace
Mod+Shift+e      → Exit i3 (with confirmation)

# Window Management
Mod+q            → Kill focused window
Mod+h/j/k/l      → Focus left/down/up/right
Mod+Left/Down/Up/Right → Same focus with arrows

Mod+Shift+h/j/k/l → Move focused window left/down/up/right
Mod+Shift+Arrows  → Same move with arrows

Mod+v            → Split container vertically
Mod+f            → Toggle fullscreen focused container
Mod+s            → Layout stacking
Mod+w            → Layout tabbed
Mod+e            → Toggle split layout

Mod+a            → Focus parent container

# Workspace Management
Mod+1..0         → Switch to workspace 1..10
Mod+Shift+1..0   → Move focused container to workspace 1..10

# Resize Mode (Mod+r)
j / Left         → Shrink width
k / Down         → Grow height
l / Up           → Shrink height
; / Right        → Grow width

Enter / Escape / Mod+r → Exit resize mode

# Volume (Hardware Keys)
XF86AudioRaiseVolume  → Volume up + refresh status
XF86AudioLowerVolume  → Volume down + refresh status
XF86AudioMute         → Toggle mute + refresh status
XF86AudioMicMute      → Toggle mic mute + refresh status

Kitty Terminal:

Ctrl+Right       → Next tab
Ctrl+Left        → Previous tab
Ctrl+Shift+Right → Move tab forward
Ctrl+Shift+Left  → Move tab backward

Ctrl+t           → New tab with current working directory
Ctrl+q / Ctrl+w  → Close tab
Ctrl+Shift+q / Ctrl+Shift+w → Close OS window

Ctrl+Shift+t     → Set tab title

Ctrl+=           → Increase font size
Ctrl+-           → Decrease font size
